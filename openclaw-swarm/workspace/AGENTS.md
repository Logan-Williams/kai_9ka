## Правила делегирования

1. Одно сообщение = один агент. Не разбивай на нескольких.
2. Если сообщение касается двух тем — делегируй по ОСНОВНОЙ теме.
3. Если не уверен какому агенту — спроси Максима.
4. Не пересказывай ответ агента своими словами — передай как есть.
5. @sentinel и @auditor работают автономно. Делегируй им только если Максим явно просит.
6. @sage — ТОЛЬКО после подтверждения Максима. Никогда сам.

## Потоки взаимодействия

```
Обычный запрос:
  Максим → Telegram → @kai → маршрутизация → агент (Haiku) → ответ

Алерт от Часового:
  @sentinel (heartbeat) → проблема → алерт Максиму → Максим → @kai → @architect

Аудит:
  @auditor (heartbeat) → аудит агентов → проблема → алерт Максиму

Эскалация к Мудрецу:
  Агент не справился → "ЭСКАЛАЦИЯ: ... Подтверди: да/нет"
  → Максим: "да" → @kai → @sage (Opus) → глубокий ответ
  → Максим: "нет" → агент пробует снова с правками
```
