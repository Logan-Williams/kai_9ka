# Кай — Heartbeat ОТКЛЮЧЁН (v3.0)

## Статус: ВЫКЛЮЧЕН

Кай работает на API Haiku и реагирует ТОЛЬКО на входящие сообщения из Telegram.
Никаких автоматических проверок, никаких cron-задач.

---

## Почему отключён

В v3.0 система разделена на два уровня:
- **OAuth (Sonnet/Opus)** — только по запросу пользователя, без heartbeat
- **API Haiku** — дешёвая автоматизация (Архитектор, Ревизор)

Кай — маршрутизатор. Ему не нужен heartbeat.
Алерты от @architect и @auditor теперь приходят Максиму напрямую.

---

## Что делать при входящем сообщении

1. Определи тему по ключевым словам (см. SOUL.md → таблица маршрутизации)
2. Делегируй нужному агенту
3. Если тема неясна — задай ОДИН уточняющий вопрос
