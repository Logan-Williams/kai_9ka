# Task Manager — Управление задачами

## Формат задач

```
[ ] Задача — @YYYY-MM-DD — !высокий — #категория
[>] Задача в работе — @YYYY-MM-DD — !средний — #работа
[x] Выполнено — @YYYY-MM-DD ✓ — #личное
[-] Отменено — причина
```

### Статусы
| Символ | Статус | Описание |
|--------|--------|----------|
| [ ] | Ожидает | Задача создана, не начата |
| [>] | В работе | Взята в работу |
| [x] | Выполнено | Завершена |
| [-] | Отменено | Отменена с причиной |

### Приоритеты
| Приоритет | Когда |
|-----------|-------|
| !высокий | Дедлайн сегодня/завтра или критичная задача |
| !средний | Нужно на этой неделе |
| !низкий | Может подождать |

### Категории
| Категория | Примеры |
|-----------|---------|
| #работа | ЧПУ, станки, производство, заказы |
| #личное | Быт, покупки, дом |
| #здоровье | Спорт, врач, обследования |
| #обучение | Курсы, книги, навыки |
| #финансы | Оплаты, счета |

## Операции

### Добавить задачу
Команда: "добавь задачу: [описание]"
Действие:
1. Записать в facts.md Ментора
2. Спросить дедлайн если не указан: "К какому числу?"
3. Назначить приоритет по умолчанию: !средний
4. Определить категорию по контексту
5. Подтвердить: "Добавил: [ ] Заказать фрезы — @2026-02-25 — !средний — #работа"

### Выполнить задачу
Команда: "сделал [задачу]" / "выполнено [задачу]"
Действие:
1. Найти задачу в facts.md
2. Изменить [ ] → [x], добавить ✓
3. Перенести в memory/YYYY-MM-DD.md
4. Подтвердить: "Отметил: [x] Заказать фрезы ✓"

### Удалить задачу
Команда: "удали [задачу]" / "отмени [задачу]"
Действие:
1. Спросить подтверждение: "Удалить '[задача]'?"
2. Если да → [-] с причиной
3. Перенести в архив

### Показать задачи
Команда: "задачи" / "список" / "что делать"
Действие: Показать активные задачи из facts.md, отсортированные по приоритету

---

## Правила

1. Максимум 3 задачи с !высокий одновременно
2. Максимум 15 активных задач всего
3. Если больше — предложить: "У тебя 16 задач. Закрыть или отменить что-то?"
4. Задачи с просроченным дедлайном — пометить: "⚠ просрочено"
5. Не добавлять/удалять без подтверждения Максима
6. Дубликаты — предупреждать: "Похожая задача уже есть: [описание]"
