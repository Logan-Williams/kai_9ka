# Daily Planner — Ежедневное планирование

## Утренний план (08:00 по Europe/Berlin)

### Формат
```markdown
## План на DD.MM (День недели)

### Топ-3 задачи дня
1. [!высокий] Задача 1
2. [!высокий/средний] Задача 2
3. [!средний] Задача 3

### Также на сегодня
- [ ] Задача 4
- [ ] Задача 5

### Дедлайны
- ⚠ [Задача X] — дедлайн сегодня
- [Задача Y] — дедлайн завтра
```

### Правила утреннего плана
1. Максимум 5 задач на день
2. Топ-3 — самые важные (начать с них)
3. Если есть просроченные задачи — показать первыми
4. Не включать задачи без конкретного действия

---

## Вечерний итог (21:00 по Europe/Berlin)

### Формат
```markdown
## Итог DD.MM

### Сделано
- [x] Задача 1 ✓
- [x] Задача 2 ✓

### Не сделано (перенести?)
- [ ] Задача 3 → завтра?
- [ ] Задача 4 → завтра?

### Завтра
- Главная задача: [описание]
```

### Правила вечернего итога
1. Показать что выполнено
2. Спросить что перенести на завтра
3. Предложить главную задачу на завтра
4. Максимум 3 предложения текста

---

## Еженедельный обзор (воскресенье)

### Формат
```markdown
## Неделя DD.MM — DD.MM

### Выполнено: N задач
- [перечень]

### Не выполнено: N задач
- [перечень с причиной]

### Паттерны
- Чаще всего откладывал: #категория
- Быстрее всего делал: #категория

### План на следующую неделю
- Топ-3 приоритета
```

---

## Интеграция с task-manager

- Утренний план берёт задачи из facts.md Ментора
- Вечерний итог обновляет статусы в facts.md
- Завершённые задачи переносятся в memory/YYYY-MM-DD.md

## Интеграция с Obsidian

При запросе "синхронизируй с Obsidian":
- Загрузи скилл obsidian-sync
- Экспортируй план дня в формате Obsidian Daily Note
