# Siemens Sinumerik 840D — Справочник фрезерной обработки

## Конфигурация Максима
- 3+2 оси (поворотный стол)
- Поддержка TRAORI (трансформация ориентации)
- ShopMill + G-код программирование

---

## Отличия от Fanuc

| Функция | Fanuc | Siemens 840D |
|---------|-------|--------------|
| Конец программы | M30 | M30 или M17 |
| Подпрограмма вызов | M98 P___ | CALL "name" или L___ |
| Радиус дуги | R / I,J,K | CR= / I,J,K |
| Переменные | #100-#999 | R0-R299 (R-параметры) |
| Коррекция длины | G43 H__ | D__ (прямое назначение) |
| Рабочие СК | G54-G59 | G54-G59 или FRAMES |
| Наклон плоскости | G68.2 | CYCLE800 / PLANE SPATIAL |
| Скругление углов | нет | RNDM= |
| Фаска | нет | CHR= / CHF= |

## G-коды (основные, отличия от ISO)

### Перемещение
| G-код | Описание |
|-------|----------|
| G0 | Ускоренное перемещение |
| G1 | Линейная интерполяция |
| G2 | Круговая по часовой |
| G3 | Круговая против часовой |
| G33 | Нарезание резьбы с постоянным шагом |

### Специфичные для Siemens
| Команда | Описание |
|---------|----------|
| RNDM=X | Скругление углов (радиус X мм) |
| CHR=X | Фаска по длине |
| CHF=X | Фаска под 45 градусов |
| BRISK | Режим точного позиционирования |
| SOFT | Плавное позиционирование |
| FFWON | Управление с опережением |

### Коррекция
| Команда | Описание |
|---------|----------|
| D1 | Активация коррекции инструмента D1 |
| G41 | Коррекция на радиус слева |
| G42 | Коррекция на радиус справа |
| G40 | Отмена коррекции |
| TOFFL | Отмена коррекции на длину |

## Стандартные циклы

### Сверление
| Цикл | Описание | Параметры |
|-------|----------|-----------|
| CYCLE81 | Сверление | RTP, RFP, SDIS, DP |
| CYCLE82 | Сверление с выдержкой | + DTB (время выдержки) |
| CYCLE83 | Глубокое сверление | + FDEP, FDPR (шаг, подача) |
| CYCLE84 | Нарезание резьбы | + SDAC, MPIT/PIT (шаг) |
| CYCLE85 | Расточка | RTP, RFP, SDIS, DP, DTB |
| CYCLE86 | Расточка с ориент. шпинделя | + SDIR, RPA, RPO |
| CYCLE89 | Расточка с выдержкой | RTP, RFP, SDIS, DP, DTB |

### Фрезерование
| Цикл | Описание |
|-------|----------|
| POCKET1 | Прямоугольный карман |
| POCKET2 | Круглый карман |
| POCKET3 | Карман произвольной формы |
| POCKET4 | Карман произвольный (с остатком) |
| SLOT1 | Продольный паз |
| SLOT2 | Круговой паз |
| LONGHOLE | Продолговатое отверстие |
| CYCLE71 | Фрезерование по контуру |
| CYCLE72 | Контур с остатком |

### Пример CYCLE83 (глубокое сверление)
```gcode
CYCLE83(50, 0, 2, -25, , 5, , 0.5, 0, 0, 1, 1, 200)
; RTP=50 (плоскость отвода)
; RFP=0 (референсная плоскость)
; SDIS=2 (безопасное расстояние)
; DP=-25 (глубина сверления)
; FDEP=5 (шаг сверления)
; DAM=0.5 (уменьшение шага)
; VRT=0 (отвод)
; AXN=1 (ось сверления)
```

## 3+2 обработка (Siemens)

### CYCLE800 (Swivel)
Основной цикл для 3+2 обработки на Siemens.

```gcode
CYCLE800(0, "TABLE", 0, 27, 0, 0, 30, 0, 0, 0, 0, 1)
; Поворот стола: A=0, B=30, C=0
; Режим 27 = TRAORI + автоматическое позиционирование
```

### TRAORI / TCARR
```gcode
TRAORI(1)         ; Включить трансформацию ориентации
TCARR=1           ; Выбрать кинематику поворотного стола
; ... обработка ...
TRAFOOF           ; Выключить трансформацию
```

### PLANE SPATIAL (альтернатива)
```gcode
PLANE SPATIAL(SPA=0, SPB=30, SPC=0, TURN)
; Наклон: A=0, B=30, C=0
; TURN = поворот осей
```

> **Важно:** CYCLE800 должен быть настроен производителем станка. Параметры зависят от кинематики.

## R-параметры

Siemens использует R-параметры вместо макро-переменных:
- R0-R99 — свободные для пользователя
- R100-R299 — могут использоваться циклами (осторожно!)

```gcode
R10 = 50.5        ; Присвоение
R11 = R10 * 2     ; Арифметика
IF R10 > 100 GOTOF LABEL1  ; Условие
```

## Типичные алярмы

| Диапазон | Тип | Примеры |
|----------|-----|---------|
| 10000-14999 | Ошибки ЧПУ | 10930: превышение диапазона перемещения |
| 15000-19999 | Ошибки циклов | 15100: ошибка в параметрах цикла |
| 20000-24999 | Ошибки компилятора | 20000: синтаксическая ошибка |
| 25000-29999 | Ошибки привода | 25000: привод не готов |
| 60000-64999 | Ошибки PLC | 60000-63999: специфичны для станка |

### Частые алярмы
| Алярм | Описание | Решение |
|-------|----------|---------|
| 10930 | Программный лимит перемещения | Проверь координаты, СК |
| 14011 | Ошибка смены инструмента | Проверь магазин, ориентацию |
| 15100 | Ошибка параметров цикла | Проверь параметры цикла |
| 17070 | CYCLE800 конфигурация | Проверь настройку CYCLE800 |
| 20000 | Синтаксическая ошибка | Проверь текст программы |
| 25000 | Привод не готов | Проверь питание приводов |

## ShopMill vs G-код

| ShopMill | G-код |
|----------|-------|
| Графический интерфейс | Текстовый |
| Без знания G-кода | Нужно знать G-код |
| Быстрая настройка простых деталей | Полный контроль |
| Ограничен стандартными операциями | Любая сложность |

> Для 3+2 обработки рекомендуется G-код — ShopMill имеет ограничения при работе с наклонными плоскостями.

## Типовая структура программы (Siemens)

```gcode
; НАЗВАНИЕ ДЕТАЛИ
; ДАТА: YYYY-MM-DD
; МАТЕРИАЛ: СТАЛЬ 45

N10 G90 G17 G54        ; Абсолютные, плоскость XY, рабочая СК
N20 T1 M6              ; Смена инструмента T1
N30 D1                  ; Коррекция D1
N40 S3000 M3            ; Обороты 3000, шпиндель CW
N50 G0 X0 Y0            ; Позиционирование
N60 G0 Z50              ; Безопасная высота
N70 M8                  ; СОЖ вкл

; ... обработка ...

N200 M9                 ; СОЖ выкл
N210 G0 Z200            ; Отвод по Z
N220 M5                 ; Стоп шпинделя
N230 M30                ; Конец программы
```
