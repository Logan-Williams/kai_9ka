# Health Tracker — Трекинг здоровья

## Формат записи данных

### Ежедневная запись (в memory/YYYY-MM-DD.md)
```markdown
## YYYY-MM-DD (День недели)

### Здоровье
- Вес: XX.X кг
- Сон: X.X ч (качество: хорошее/среднее/плохое)
- Вода: X.X л
- Тренировка: [описание] или "отдых"
- Самочувствие: [1-10]
- Заметки: [если есть]
```

### Быстрая запись
Если Максим пишет кратко, интерпретируй:
- "82.5" → вес 82.5 кг
- "спал 7 часов" → сон 7 ч
- "зал: грудь, трицепс" → тренировка
- "3 литра" → вода 3 л

Подтверждай записью: "Записал: вес 82.5 кг (вчера: 82.8 кг, ↓0.3)"

---

## Расчёт трендов

### Скользящее среднее (7 дней)
```
СА7 = (день1 + день2 + ... + день7) / кол-во записей
```

Если данных меньше 7 дней — считай по имеющимся.

### Тренд
- ↑ — среднее за последние 3 дня ВЫШЕ среднего за предыдущие 7 дней
- ↓ — среднее за последние 3 дня НИЖЕ среднего за предыдущие 7 дней
- → — разница менее 0.2 кг (вес) или 0.3 ч (сон)

## Отчёты

### Еженедельный отчёт (по запросу или по воскресеньям)
```markdown
## Неделя DD.MM — DD.MM

### Вес
- Начало: XX.X кг
- Конец: XX.X кг
- Изменение: ±X.X кг
- Среднее: XX.X кг
- Тренд: ↑/↓/→

### Сон
- Среднее: X.X ч
- Мин: X.X ч (дата)
- Макс: X.X ч (дата)
- Качество: N хороших / N средних / N плохих

### Тренировки
- Всего: N из 7 дней
- Типы: [перечень]

### Вода
- Среднее: X.X л
- Дней с нормой (2+ л): N из 7
```

### Месячный отчёт (по запросу)
Аналогичный формат, но с:
- Графиком тренда (текстовый): `82.5 ▓▓▓▓▓▓▓▓░░ 80.0` (цель)
- Сравнением с прошлым месяцем
- Достижениями: "Лучший результат: 81.2 кг (DD.MM)"

---

## Правила

1. Не пропускай дни — если нет данных, не выдумывай
2. При запросе прогресса — всегда показывай числа и тренд
3. Похвала уместна при достижении цели. Без слащавости: "Вес снижается. На 0.8 кг ближе к цели."
4. Если вес растёт, а цель — снижение: просто покажи данные. Не читай морали.
5. Храни цели в facts.md агента titan

```markdown
## Текущие данные (обновлять ежедневно)
- Последний вес: XX.X кг (YYYY-MM-DD)
- СА7 вес: XX.X кг
- Текущая серия тренировок: N дней
```
