# Fanuc Oi-MF Plus — Справочник фрезерной обработки

## Конфигурации Максима
- **3+1**: 3 линейных оси (X, Y, Z) + 1 поворотная (A или B)
- **3+2**: 3 линейных + 2 поворотных (A+C или B+C), поворотный стол, поддержка RTCP

---

## G-коды (основные)

### Перемещение
| G-код | Описание | Модальный |
|-------|----------|-----------|
| G00 | Ускоренное перемещение (rapid) | Да |
| G01 | Линейная интерполяция (рабочая подача) | Да |
| G02 | Круговая интерполяция по часовой | Да |
| G03 | Круговая интерполяция против часовой | Да |

### Плоскости
| G-код | Описание |
|-------|----------|
| G17 | Плоскость XY (для фрезерования в Z) |
| G18 | Плоскость ZX |
| G19 | Плоскость YZ |

### Системы координат
| G-код | Описание |
|-------|----------|
| G54-G59 | Рабочие системы координат (1-6) |
| G54.1 P1-P48 | Дополнительные рабочие системы координат |
| G53 | Машинная система координат (немодальный) |

### Референсные точки
| G-код | Описание |
|-------|----------|
| G28 | Возврат в референсную точку (через промежуточную) |
| G30 | Возврат во 2-ю/3-ю/4-ю референсную точку |
| G91 G28 Z0 | Безопасный вывод Z в реф. точку |

### Коррекция инструмента
| G-код | Описание |
|-------|----------|
| G40 | Отмена коррекции на радиус |
| G41 | Коррекция на радиус слева |
| G42 | Коррекция на радиус справа |
| G43 | Коррекция на длину (+) |
| G44 | Коррекция на длину (-) |
| G49 | Отмена коррекции на длину |

### Режимы
| G-код | Описание |
|-------|----------|
| G90 | Абсолютное программирование |
| G91 | Инкрементальное программирование |
| G94 | Подача в мм/мин |
| G95 | Подача в мм/об |
| G98 | Возврат на уровень начальной точки (циклы) |
| G99 | Возврат на уровень R (циклы) |

## M-коды (основные)

| M-код | Описание |
|-------|----------|
| M00 | Программный останов |
| M01 | Условный останов (если кнопка OPT STOP включена) |
| M02 | Конец программы |
| M03 | Шпиндель по часовой (CW) |
| M04 | Шпиндель против часовой (CCW) |
| M05 | Стоп шпинделя |
| M06 | Смена инструмента |
| M08 | Включить СОЖ |
| M09 | Выключить СОЖ |
| M19 | Ориентация шпинделя |
| M30 | Конец программы и перемотка |
| M98 | Вызов подпрограммы |
| M99 | Возврат из подпрограммы |

> M-коды выше M30 зависят от конкретного станка. Проверяй в документации.

## Циклы сверления

| G-код | Цикл | Применение |
|-------|-------|------------|
| G73 | Прерывистое сверление (chip-breaking) | Глубокие отверстия, отвод стружки |
| G74 | Нарезание левой резьбы метчиком | Левая резьба |
| G76 | Расточка с ориентацией | Точные отверстия |
| G80 | Отмена цикла сверления | |
| G81 | Сверление (без выдержки) | Неглубокие отверстия |
| G82 | Сверление с выдержкой | Зенковка, точные глубины |
| G83 | Глубокое сверление (пек-дриллинг) | Глубокие отверстия с полным выводом |
| G84 | Нарезание резьбы метчиком | Правая резьба |
| G85 | Расточка (вывод на подаче) | Чистовая расточка |
| G86 | Расточка (вывод на быстром, шпиндель стоп) | Черновая расточка |

### Пример цикла сверления
```gcode
G90 G54 G17          (абсолютные координаты, рабочая СК, плоскость XY)
G00 X50. Y30.        (позиционирование над отверстием)
G43 H01 Z50.         (коррекция на длину, безопасная высота)
G83 Z-25. R3. Q5. F200 (глубокое сверление: глубина -25, R-плоскость 3, шаг 5мм, подача 200)
X80. Y30.             (следующее отверстие)
X80. Y60.             (ещё одно)
G80                    (отмена цикла)
G91 G28 Z0            (домой по Z)
```

## 3+2 обработка (Fanuc)

### Основные понятия
- **RTCP (Rotary Tool Center Point)**: компенсация поворота инструмента при наклоне осей
- **G68.2**: наклон рабочей плоскости (Tilted Working Plane)
- **G69**: отмена наклона

### Пример 3+2 с G68.2
```gcode
G90 G54 G17
G69                    (отмена предыдущего наклона)
G68.2 X0 Y0 Z0 I90 J0 K0  (наклон: поворот вокруг X на 90 градусов)
G54.1 P1               (или отдельная СК для наклонной плоскости)
G43.4 H01              (коррекция на длину с RTCP)
(... обработка в наклонной плоскости ...)
G49                     (отмена коррекции)
G69                     (отмена наклона)
```

> **Важно:** Точная настройка G68.2 зависит от конфигурации станка (расположение осей вращения). Проверяй параметры станка перед использованием.

## Типичные алярмы

### PS (Program Error)
| Алярм | Описание | Решение |
|-------|----------|---------|
| PS0001 | Ошибка формата TH-кода | Проверь формат блока |
| PS0003 | Слишком много цифр | Проверь значения координат |
| PS0010 | G-код не используется | Проверь правильность G-кода |
| PS0015 | Слишком много осей в одном блоке | Разбей на несколько блоков |
| PS0059 | G68.2 ошибка | Проверь параметры наклона |

### SV (Servo Error)
| Алярм | Описание | Решение |
|-------|----------|---------|
| SV0401 | Ошибка серво V-ready off | Проверь привод, перезапусти |
| SV0410 | Перегрузка серво | Снизь подачу/ускорение |
| SV0447 | Ошибка энкодера | Проверь кабели энкодера |

### OT (Overtravel)
| Алярм | Описание | Решение |
|-------|----------|---------|
| OT0500 | Превышение программного лимита + | Проверь координаты |
| OT0501 | Превышение программного лимита - | Проверь координаты |
| OT0510 | Сработал концевик + | Выведи ось в ручном режиме |
| OT0511 | Сработал концевик - | Выведи ось в ручном режиме |

> Для полного списка алярмов используй руководство оператора Fanuc Oi-MF Plus.

## Типовая структура программы

```gcode
O0001 (НАЗВАНИЕ ДЕТАЛИ)
(ДАТА: YYYY-MM-DD)
(МАТЕРИАЛ: СТАЛЬ 45)
(ИНСТРУМЕНТ: T01 - ФРЕЗА D10)

N10 G91 G28 Z0         (домой по Z)
N20 T01 M06             (смена инструмента)
N30 G90 G54 G17         (абсолютные, рабочая СК, плоскость XY)
N40 S3000 M03           (обороты 3000, шпиндель по часовой)
N50 G00 X0 Y0           (позиционирование)
N60 G43 H01 Z50.        (коррекция на длину)
N70 M08                 (включить СОЖ)

(... обработка ...)

N200 M09                (выключить СОЖ)
N210 G91 G28 Z0         (домой по Z)
N220 G28 X0 Y0          (домой по XY)
N230 M05                (стоп шпинделя)
N240 M30                (конец программы)
```
