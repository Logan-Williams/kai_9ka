# Obsidian Sync — Синхронизация заметок

## Структура папок в Obsidian

```
Vault/
├── Daily/              # Ежедневные заметки
│   └── YYYY-MM-DD.md
├── Tasks/              # Задачи и планы
│   ├── active.md       # Текущие задачи
│   └── archive/        # Завершённые
├── CNC/                # Заметки по ЧПУ
│   ├── programs/       # Программы и их особенности
│   ├── problems/       # Решённые проблемы
│   └── reference/      # Справочные заметки
├── Health/             # Здоровье
│   ├── weight-log.md   # Лог веса
│   ├── workouts/       # Тренировки
│   └── reports/        # Отчёты
└── Templates/          # Шаблоны
    ├── daily.md
    └── problem-solved.md
```

---

## Формат ежедневной заметки

```markdown
---
date: YYYY-MM-DD
day: Понедельник
tags: [daily]
---

# DD.MM.YYYY — День недели

## План дня
- [ ] Задача 1
- [ ] Задача 2
- [ ] Задача 3

## Здоровье
- Вес: XX.X кг
- Сон: X ч
- Тренировка: —

## Заметки
-

## Итог дня
- Выполнено: N/N задач
```

---

## Формат заметки о решённой проблеме (CNC)

```markdown
---
date: YYYY-MM-DD
tags: [cnc, problem, решено]
станок: Fanuc Oi-MF Plus
---

# Проблема: [краткое описание]

## Симптомы
- [что наблюдалось]

## Причина
- [что оказалось причиной]

## Решение
- [что сделали]

## Примечания
- [что учесть в будущем]
```

## Правила именования файлов

| Тип | Формат имени | Пример |
|-----|-------------|--------|
| Ежедневная | YYYY-MM-DD.md | 2026-02-22.md |
| Задачи | active.md | active.md |
| ЧПУ проблема | problem-YYYY-MM-DD-описание.md | problem-2026-02-22-vibration-fanuc.md |
| Отчёт здоровья | report-YYYY-WNN.md | report-2026-W08.md |

## Синхронизация

### При запросе "синхронизируй с Obsidian"
1. Сформируй заметку в формате выше
2. Сохрани в workspace (memory/) — Obsidian Sync подхватит автоматически
3. Подтверди: "Заметка сохранена: Daily/2026-02-22.md"

### При запросе "что нового в Obsidian"
1. Прочитай последние файлы в memory/
2. Покажи новые/изменённые заметки

## YAML Frontmatter

Всегда добавляй YAML frontmatter к заметкам:
- `date` — дата создания
- `tags` — теги для поиска
- Дополнительные поля по типу заметки

Теги используй на русском с транслитерацией для технических:
- `daily`, `weekly`, `cnc`, `health`, `task`, `problem`, `решено`
