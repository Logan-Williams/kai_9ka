# Heidenhain TNC 640 — Справочник фрезерной обработки

## Конфигурация Максима
- 3+2 оси (поворотный стол)
- Klartext (разговорное) программирование
- PLANE SPATIAL для наклонных плоскостей
- Touch probe циклы

---

## Klartext — Разговорное программирование

Heidenhain использует текстовый формат, отличный от стандартного G-кода.

### Основные команды перемещения

| Команда | Описание | G-код аналог |
|---------|----------|--------------|
| L X+50 Y+30 R0 FMAX M3 | Линейное перемещение на быстром | G0 X50 Y30 M3 |
| L X+50 Y+30 R0 F500 | Линейное перемещение с подачей | G1 X50 Y30 F500 |
| CC X+0 Y+0 | Центр дуги | |
| C X+50 Y+0 DR+ | Дуга по часовой | G2 |
| C X+50 Y+0 DR- | Дуга против часовой | G3 |

### Коррекция на радиус
| Код | Описание |
|-----|----------|
| RL | Коррекция слева от контура |
| RR | Коррекция справа от контура |
| R0 | Без коррекции (центр инструмента) |

```

  Q204=50  ;2-Е БЕЗОП. РАССТОЯНИЕ~
  Q212=0   ;СЪЕМ ПРИ ВОЗВРАТЕ~
  Q205=3   ;МИН. ГЛУБИНА ВРЕЗАНИЯ
```

## PLANE SPATIAL — 3+2 обработка

Основная команда для наклона рабочей плоскости на TNC 640.

### Синтаксис
```
PLANE SPATIAL SPA+0 SPB+30 SPC+0 TURN FMAX SEQ- TABLE ROT
```

| Параметр | Описание |
|----------|----------|
| SPA | Угол поворота вокруг A |
| SPB | Угол поворота вокруг B |
| SPC | Угол поворота вокруг C |
| TURN | Режим: поворот осей |
| FMAX | Быстрый поворот |
| SEQ- | Последовательность: сначала основная ось |
| TABLE ROT | Поворачивать стол (не шпиндель) |

### Варианты команды PLANE
| Команда | Описание |
|---------|----------|
| PLANE SPATIAL | По углам Эйлера (SPA, SPB, SPC) |
| PLANE PROJECTED | По проекциям |
| PLANE EULER | По углам Эйлера (другая конвенция) |
| PLANE VECTOR | По вектору нормали |
| PLANE RESET | Сброс наклона |

### Пример 3+2 обработки
```
TOOL CALL 1 Z S3000 F500    ; Вызов инструмента
L Z+200 R0 FMAX             ; Безопасная высота
PLANE SPATIAL SPA+0 SPB+30 SPC+0 TURN FMAX SEQ- TABLE ROT
; Стол повёрнут на 30 градусов по B
L X+0 Y+0 R0 FMAX M3        ; Позиционирование
L Z+50 R0 FMAX               ; Подвод к детали
; ... обработка в наклонной плоскости ...
L Z+200 R0 FMAX M5           ; Отвод
PLANE RESET TURN FMAX SEQ-   ; Сброс наклона
```

## Q-параметры

| Тип | Диапазон | Описание |
|-----|----------|----------|
| Q | Q0-Q99 | Числовые параметры пользователя |
| Q | Q100-Q199 | Используются циклами (осторожно!) |
| QS | QS0-QS99 | Строковые параметры |
| QL | QL0-QL99 | Локальные (только в текущей программе) |

```
Q1 = 50.5           ; Присвоение
Q2 = Q1 * 2         ; Арифметика
FN 1: Q3 = +Q1 + +Q2  ; Сложение (старый формат)
```

## Таблица инструмента

Heidenhain хранит данные инструмента в файлах:
- **TOOL.T** — основная таблица
- **TOOLP.TRN** — расширенные данные

### Основные поля
| Поле | Описание |
|------|----------|
| T | Номер инструмента |
| L | Длина |
| R | Радиус |
| R2 | Радиус скругления |
| CUT | Количество зубьев |
| LCUTS | Длина режущей части |

### Вызов инструмента
```
TOOL CALL 1 Z S3000       ; Инструмент T1, ось Z, 3000 об/мин
TOOL CALL 5 Z S5000 F800  ; T5, 5000 об/мин, подача 800
```

## Типичные ошибки

| Ошибка | Описание | Решение |
|--------|----------|---------|
| FE "контроль контура" | Ошибка контура (конечная точка) | Проверь координаты точек |
| FE "CLAMP" | Ось зажата | Разожми ось вручную |
| FE "перемещение" | Программный лимит | Проверь координаты |
| FE "инструмент" | Инструмент не определён | Проверь таблицу TOOL.T |
| FE "PLANE" | Ошибка наклона плоскости | Проверь углы PLANE SPATIAL |

> Коды ошибок Heidenhain отличаются по станкам. Точный текст ошибки важнее кода.

## Типовая структура программы

```
0  BEGIN PGM DETALNAME MM
1  ; НАЗВАНИЕ: Деталь
2  ; ДАТА: YYYY-MM-DD
3  ; МАТЕРИАЛ: Сталь 45
4  BLK FORM 0.1 Z X-50 Y-50 Z-30   ; Заготовка (мин)
5  BLK FORM 0.2 X+50 Y+50 Z+0      ; Заготовка (макс)
6  TOOL CALL 1 Z S3000               ; Вызов T1
7  L Z+200 R0 FMAX M3               ; Безопасная высота
8  L X+0 Y+0 R0 FMAX                ; Позиционирование
9  L Z+2 R0 FMAX                    ; Подвод
10 ; ... обработка ...
11 L Z+200 R0 FMAX M9               ; Отвод, СОЖ выкл
12 L X+0 Y+0 R0 FMAX M5             ; Домой, шпиндель стоп
13 END PGM DETALNAME MM
```
