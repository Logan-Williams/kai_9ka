# Устранение проблем — Фрезерная обработка

## Вибрация (дребезг)

### Причины и решения

| Причина | Признаки | Решение |
|---------|----------|---------|
| Большой вылет инструмента | Характерный звук, следы на поверхности | Уменьшить вылет, использовать оправку с термозажимом |
| Высокие обороты | Резонанс | Снизить обороты на 10-20% или наоборот поднять (выйти из резонанса) |
| Большая глубина резания (ap) | Нестабильный процесс | Уменьшить ap, увеличить количество проходов |
| Большая ширина резания (ae) | Повышенная нагрузка | Уменьшить ae до 30-40%D |
| Слабый зажим детали | Вибрация всей детали | Усилить зажим, добавить подпорки |
| Изношенный инструмент | Нарастающая вибрация | Заменить инструмент |
| Тонкая стенка детали | Вибрация стенки | Снизить ap, попутное фрезерование, подпорка |

### Чеклист при вибрации
1. [ ] Проверить вылет инструмента (макс 4×D)
2. [ ] Проверить зажим детали (люфтов нет?)
3. [ ] Попробовать другие обороты (±15-20%)
4. [ ] Уменьшить ap и ae
5. [ ] Проверить износ инструмента
6. [ ] Переключить на попутное фрезерование (если встречное)
7. [ ] Проверить жёсткость оправки

## Плохая шероховатость

### Причины и решения

| Причина | Решение |
|---------|---------|
| Тупой инструмент | Заменить |
| Высокая подача на зуб | Снизить fz |
| Низкие обороты | Увеличить обороты (повысить Vc) |
| Вибрация | См. раздел "Вибрация" |
| Нарост на режущей кромке (BUE) | Увеличить Vc, использовать инструмент с покрытием |
| Плохой отвод стружки | Улучшить подачу СОЖ, увеличить давление |
| Неправильное направление фрезерования | Переключить попутное/встречное |

### Чеклист для чистовой обработки
1. [ ] Vc на верхней границе рекомендаций
2. [ ] fz снижена (0.04-0.08 мм/зуб для стали)
3. [ ] ap малая (0.1-0.3 мм)
4. [ ] Инструмент острый (новый или слабо изношенный)
5. [ ] Стружка отводится нормально
6. [ ] Попутное фрезерование (для чистоты поверхности)
7. [ ] Нет вибрации

## Сколы на кромках

| Причина | Решение |
|---------|---------|
| Выход инструмента через кромку | Фрезеровать "от кромки", а не "к кромке" |
| Хрупкий материал (чугун, закалённая сталь) | Снизить подачу на выходе, оставить припуск |
| Слишком острый инструмент | Использовать фрезу с фаской на кромке |
| Большая подача | Снизить fz при выходе |

## Увод размера

### Причины

| Причина | Диагностика | Решение |
|---------|-------------|---------|
| Температурное расширение | Размер "плывёт" в течение дня | Прогреть станок, компенсация в программе |
| Износ инструмента | Размер постепенно уходит | Корректор износа (G41/G42 + H/D корректор) |
| Отжим инструмента | Размер больше чем надо | Уменьшить нагрузку, жёстче оправка |
| Неточность СК | Все размеры сдвинуты | Проверить нулевую точку |
| Люфт оси | Размер зависит от направления | Компенсация люфта в параметрах |

### Компенсация износа
```
Fanuc:  H или D корректор → Offset → Wear
Siemens: D корректор → $TC_DP3 (длина) / $TC_DP6 (радиус)
Heidenhain: TOOL.T → DL (износ длины) / DR (износ радиуса)
```

## Поломка инструмента

### Анализ причин

| Характер поломки | Вероятная причина |
|------------------|-------------------|
| Сломан на входе | Удар, слишком быстрый подвод (G0 вместо G1) |
| Сломан в процессе | Перегрузка (ap/ae/fz слишком большие) |
| Скол режущей кромки | Вибрация, прерывистое резание, удар по корке |
| Сломан у хвостовика | Биение, неправильный зажим |
| Постепенный износ → поломка | Не заменён вовремя |

### Профилактика
1. Контролируй износ (визуально или по звуку/нагрузке)
2. Не превышай рекомендуемые режимы
3. Проверяй биение инструмента (макс 0.01-0.02 мм)
4. На входе в материал — снижай подачу на 50%
5. При прерывистом резании — снижай fz на 20-30%

## Станок не выходит в ноль

### Чеклист

| Шаг | Проверка |
|-----|----------|
| 1 | Батарейка энкодера (абсолютные датчики) — заменить если севшая |
| 2 | Кабели энкодера — визуальный осмотр, нет ли повреждений |
| 3 | Концевики — не заклинены ли механически |
| 4 | Алярм серво — прочитай алярм, запиши номер |
| 5 | Параметры реф. точки — не сбились ли (проверь в параметрах) |
| 6 | Ручной вывод — попробуй вывести ось вручную (MPG) |

## Ошибки при 3+2 обработке

### Общие проблемы

| Проблема | Причина | Решение |
|----------|---------|---------|
| Деталь "сдвигается" после поворота | Неправильная точка поворота (pivot point) | Калибровать точку поворота кинематики |
| Контур смещён | Нулевая точка не пересчитана | Проверить СК после поворота |
| Станок врезается в деталь | RTCP/TRAORI не активирован | Включить компенсацию (G43.4 / TRAORI / PLANE SPATIAL) |
| Углы поворота неправильные | Неправильный порядок осей | Проверить последовательность углов |
| Ось не поворачивается | Ось зажата (clamp) | Разжать ось, проверить давление |

### Калибровка точки поворота
```
Fanuc:  параметры #5401-#5406 (смещения осей вращения)
Siemens: $MC_TRAFO_AXES, $MC_TRAFO_GEOAX
Heidenhain: MOD → KinematicsOpt (автоматическая калибровка)
```

## Проблемы с СОЖ

| Проблема | Решение |
|----------|---------|
| Нет давления | Проверить насос, фильтр, уровень СОЖ |
| Плохой запах | Заменить СОЖ, проверить концентрацию |
| Пена | Снизить давление, проверить концентрацию |
| Ржавчина на деталях | Повысить концентрацию, проверить pH |
| Раздражение кожи | Проверить концентрацию (не >12%), заменить |

### Рекомендуемая концентрация
- Сталь: 5-8%
- Нержавейка: 8-12%
- Алюминий: 5-8%
- Чугун: без СОЖ (сухое)

## Признаки износа инструмента

| Признак | Что делать |
|---------|------------|
| Блестящая полоска на задней грани | Износ нормальный, контролируй ширину (>0.3мм = заменить) |
| Скол режущей кромки | Заменить немедленно |
| Нарост на кромке (BUE) | Увеличить скорость, покрытие |
| Кратерный износ (лунка на передней грани) | Снизить скорость или заменить |
| Увеличение усилия резания | Скоро поломка, заменить |
| Изменение звука (визг, скрежет) | Износ или вибрация, проверить |
| Ухудшение шероховатости | Инструмент тупой, заменить |
