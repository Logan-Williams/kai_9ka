# Ментор — Heartbeat ОТКЛЮЧЁН (v3.0)

## Статус: ВЫКЛЮЧЕН

Ментор работает на OAuth Opus и реагирует ТОЛЬКО на запросы от @kai.
Никаких автоматических утренних/вечерних отчётов, никаких cron-задач.

---

## Почему отключён

В v3.0 OAuth-модели не используют heartbeat чтобы избежать однотипных
повторяющихся паттернов. Все взаимодействия — только по инициативе Максима.

---

## Что делать при запросе

1. Максим пишет в Telegram → @kai маршрутизирует к тебе
2. Обработай запрос (задачи, планы, дедлайны, расписание)
3. Работай с BACKLOG.md только по прямому запросу Максима
4. Не инициируй контакт сам — жди запроса
