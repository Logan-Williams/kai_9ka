# Heartbeat — Ревизор (каждые 720 мин = 12 часов, API Haiku)

> Ревизор работает на API Claude Haiku 4.5.
> Стоимость: ~$0.05/день. Нужны мозги чтобы ловить имитацию и враньё.

## Чеклист (выполняй по порядку)

### 1. Загрузи скилл
```
Прочитай: skills/agent-audit/SKILL.md
```

### 2. Проверь честность каждого агента
Для каждого из: @architect, @mentor, @titan, @technologist, @kai:
```bash
ls -lt ~/.openclaw/agents/[агент]/memory/ | head -3
```
- Прочитай последние 3 записи
- Сравни: заявленное действие vs реальный результат
- Есть ли файлы/данные которые агент "создал"?

### 3. Детекция имитации работы
Для каждого агента с heartbeat (только @sentinel в v3.1):
```bash
# Сравни последние 3 записи — есть ли разница?
diff <(head -20 ~/.openclaw/agents/sentinel/memory/[файл1]) <(head -20 ~/.openclaw/agents/sentinel/memory/[файл2])
```
- Если diff пустой или различия минимальны → ⚠️ подозрение
- Если 3 записи почти одинаковые → ❌ имитация

### 4. Проверь OAuth-агентов (по запросу)
OAuth-агенты (Технолог, Титан, Ментор) не имеют heartbeat, поэтому проверяй:
- Есть ли записи в memory/ за последние 24ч?
- Если были запросы — адекватны ли ответы?
- Не было ли ошибок при обработке?

### 5. Проверь логи на ошибки и loop detection
```bash
journalctl -u openclaw --since "12 hours ago" | grep -i "loop\|error\|repeated\|stuck"
```

### 6. Запиши результат
- Всё OK → запиши `АУДИТ_ОК — YYYY-MM-DD HH:MM` в свой memory/
- Проблемы → [АЛЕРТ] Максиму с деталями

## Правило

НЕ беспокой Максима если всё в порядке. Только АУДИТ_ОК в memory/.
Проблемы → [АЛЕРТ] НЕМЕДЛЕННО.
