# Heartbeat — Часовой (каждые 30 мин)

## Шаг 1: Проверь memory/

Прочитай свой memory/. Если уже отправлял алерт и он не решён → запиши "Ждём" → СТОП.

## Шаг 2: Проверь сервер

Запусти ЭТУ КОМАНДУ (копируй как есть):
```bash
echo "DISK=$(df -h / | tail -1 | awk '{print $5}' | tr -d '%') RAM=$(free -m | grep Mem | awk '{print int($3/$2*100)}') GW=$(systemctl is-active openclaw 2>/dev/null || echo unknown)"
```

Прочитай результат. Пример: `DISK=45 RAM=62 GW=active`

- DISK больше 80? → Отправь: `[ЧАСОВОЙ] Диск DISK%. Напиши "разберись с диском"`
- RAM больше 90? → Отправь: `[ЧАСОВОЙ] RAM RAM%. Напиши "разберись с памятью"`
- GW не active? → Отправь: `[КРИТИЧЕСКИЙ] Gateway упал! Проверь сервер!`

## Шаг 3: Проверь ошибки

Запусти ЭТУ КОМАНДУ:
```bash
journalctl -u openclaw --since "30 min ago" --no-pager 2>/dev/null | grep -ic "error\|fatal\|crash" || echo 0
```

Результат — число. Если больше 0 → Отправь: `[ЧАСОВОЙ] N ошибок за 30 мин. Напиши "проверь логи"`

## Шаг 4: Проверь время

Запусти:
```bash
TZ='Europe/Berlin' date '+%H'
```

Результат — час (число от 00 до 23).

- Час = 07 или 08? Проверь memory/ — уже отправлял утреннее сегодня? Если нет → Отправь: `Доброе утро! Напиши "план на сегодня"`
- Час = 12 или 13? Проверь memory/ — уже отправлял обеденное? Если нет → Отправь: `Обед! Напиши "вес" если взвешивался`
- Час = 21 или 22? Проверь memory/ — уже отправлял вечернее? Если нет → Отправь: `Вечер! Напиши "итоги дня". Тренировка?`

## Шаг 5: Бэкап (раз в день)

Запусти:
```bash
ls ~/.openclaw-backup-$(date +%F).tar.gz 2>/dev/null && echo YES || echo NO
```

Если NO → запусти:
```bash
tar czf ~/.openclaw-backup-$(date +%F).tar.gz ~/.openclaw/ && find ~/ -maxdepth 1 -name ".openclaw-backup-*.tar.gz" -mtime +7 -delete && echo DONE
```

## Шаг 6: Запиши результат

Запиши в memory/:
```
ОБХОД YYYY-MM-DD HH:MM — Диск:XX% RAM:XX% GW:active Ошибки:0 Алерты:нет
```

Если не отправлял никаких алертов и напоминаний → всё. ОБХОД_ОК. Молчи.
