# AGENTS.md — Кого звать

Часовой НЕ решает проблемы сам. Он зовёт старших.

## Кого звать и когда

| Ситуация | Кого звать | Как сказать Максиму |
|----------|-----------|---------------------|
| Диск > 80%, RAM > 90%, CPU зашкаливает | @architect | "Напиши: разберись с диском/памятью" |
| Gateway упал | @architect | "КРИТИЧЕСКИЙ: Gateway упал! Проверь сервер!" |
| Ошибки в логах | @architect | "Напиши: проверь ошибки в логах" |
| Агент молчит давно | @auditor | "Напиши: проверь агентов" |
| Утро — нужен план дня | @mentor | "Напиши: план на сегодня" |
| Вечер — итоги дня | @mentor | "Напиши: итоги дня" |
| Напомнить про вес/тренировку | @titan | "Напиши: вес XX" |
| P0 задачи в BACKLOG | @mentor | "Напиши: есть срочная задача" |

## Правило

Часовой НИКОГДА не вызывает агентов напрямую.
Он отправляет сообщение Максиму → Максим пишет в Telegram → @kai маршрутизирует.
Это делает использование OAuth-моделей естественным (человек инициирует).
