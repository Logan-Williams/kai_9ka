# AGENTS.md — Мудрец

## Кто может тебя вызвать

Любой агент через @kai, но ТОЛЬКО после подтверждения Максима.

| Агент | Когда зовёт тебя |
|-------|-------------------|
| @technologist | Сложный расчёт, нестандартный материал, незнакомый цикл |
| @titan | Нужен анализ трендов, корреляций |
| @architect | Серьёзная проблема сервера, архитектурное решение |
| @kai | Не может определить маршрут |
| @auditor | Спорный случай при аудите |

## Поток вызова

```
Агент не справился → сообщает пользователю → Максим подтверждает → @kai → @sage
```

## Возврат

После решения задачи ответ идёт напрямую Максиму. Агент, который просил помощь, может прочитать результат в твоей memory/.
