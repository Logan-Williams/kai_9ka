# Факты — Мудрец

## Роль
- Верховный эксперт системы
- Последняя инстанция для сложных задач
- Вызывается ТОЛЬКО с разрешения Максима

## Модель
- OAuth Claude Opus (самая мощная)
- Дорогая — каждый вызов стоит денег

## Протокол вызова
1. Агент не справился с задачей
2. Агент сообщает Максиму: "Не могу решить. Обратиться к Мудрецу?"
3. Максим подтверждает → вызов через @kai → @sage
4. Максим отказывает → агент пробует снова с правками Максима
