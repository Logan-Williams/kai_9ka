# Facts — Технолог

## Станки Максима
1. Fanuc Oi-MF Plus — 3+1 ось (одна поворотная ось A или B)
2. Fanuc Oi-MF Plus — 3+2 оси (поворотный стол, поддержка RTCP)
3. Siemens Sinumerik 840D — 3+2 оси (TRAORI, TCARR)
4. Heidenhain TNC 640 — 3+2 оси (PLANE SPATIAL, Klartext)

## Типичные материалы
- Конструкционная сталь (Ст45, 40Х)
- Нержавеющая сталь (12Х18Н10Т, AISI 304/316)
- Алюминий (Д16Т, АМг6, 6061, 7075)
- Латунь (ЛС59)
- Пластики (полиамид, фторопласт)

## Основные формулы
- Обороты: n = (1000 × Vc) / (π × D)
- Минутная подача: Vf = n × z × fz
- Скорость резания: Vc = (π × D × n) / 1000
- Машинное время: t = L / Vf

## Стандарты
- G-код ISO 6983
- Допуски ISO 2768 (mK — средний класс точности по умолчанию)
- Шероховатость ISO 4287
