# OpenClaw Swarm v4.0 — Максим Лебедев

7 AI-агентов для ЧПУ, здоровья, мониторинга, аудита + Мудрец (Opus) для сложных задач.

## Быстрый старт (деплой через Git)

```bash
# 1. На VPS — клонируй репо
git clone https://github.com/ТВОЙ_ЮЗЕР/openclaw-swarm.git
cd openclaw-swarm

# 2. Заполни ключи
cp .env.example .env
nano .env

# 3. Деплой (установит OpenClaw, скопирует файлы, запустит)
bash deploy.sh

# 4. Подключи OAuth (для агентов на Claude)
openclaw auth login

# 5. Проверь
openclaw status
```

**Обновление после изменений:**
```bash
cd openclaw-swarm && git pull && bash deploy.sh
```

## Эскалация к Мудрецу

Агент не справился → спрашивает у Максима разрешение → Максим подтверждает → Мудрец (Opus) решает задачу.
